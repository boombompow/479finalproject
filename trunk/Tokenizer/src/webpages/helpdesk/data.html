Disk quota applies to each segment of disk space, and it is not transferable between segments. To access your disk space from Windows, use your ENCS account to map the network drive. In Unix/Linux, simply change to the directory where the disk space is located by using the "cd" command. The disk space to which you are entitled can be determined using the following table.

Notes:

1. Basic Space: This is the default disk space assigned when your personal account is created. It is in three locations: the Windows home (drive G), the Unix home (drive U) and the Web directory (drive W). The disk quota for this space is established in function of your status in ENCS, and cannot be increased.

2. Thesis Space: All MASC and PHD graduate students get 500MB of extra space for their research work. Full-time faculty members who are supervising thesis students get an additional 1GB of space, which is not transferable to students.

3. Disk Space Used For Research: The disk space for research purpose should be allocated from the professor's private space(300GB so far)

4. Admin /Role Account Space: This space exists for the purpose of administration support. Thus, it is available upon request to a departmental chair, departmental administrator, and so on. The "account" associated with this space cannot be used for interactive logins to any ENCS system. It serves for sharing files among the group members (possibly only one person) and for e-mail if desired.

5. Course Support Space: This space contains the course material created the instructor. The "account" associated with this space cannot be used for interactive logins. The files saved in this space are archived after the end of each term and made available to the other professors in the same department, unless the current instructor decides to remove his/her course material.

6. Course Project Space: This space is used by students who require extra space for a course project. It is created for the students upon the request of the course instructor. The group "account" associated with this space cannot be used for interactive logins. Once the course is finished, this space is deleted. A course project account can be assigned to a single student, or be shared among several students working cooperatively.

7. Research Group Space: This space is made available to research groups in order to allow a basic presence within the ENCS structure (for example a web page to represent the group, filesharing space for cooperative work on documents, and so on). The space is in the standard group area and the professors can get 1 GB of space as a default. Any request more than that needs to be negotiated.

8. INBOX : Default incoming mail spool (Inbox), 76 MB quota (no exceptions). This space is usually accessed via the IMAP protocol (a mail reading agent or webmail). Direct access is possible only from the host login.encs.concordia.ca.

Backups of Your Files

All ENCS users can access the on-line backup system, which creates 'snapshots' (read-only copies of your files) at regular intervals. You can recover files that have been deleted or modified by copying files from the snapshot directory to your home directory.

NOTE: Only the network shares are backed up by the system. The files in local drive, such as drive C:, are NOT backed up. Once they are gone, they are gone.
