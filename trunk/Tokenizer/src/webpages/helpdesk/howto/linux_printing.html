Printing From a Linux User-Managed Machine

IMPORTANT

You must be logged in as root to setup the printer. Part of the setup process requires you to enter your ENCS username and password such that it can be sent to the print server whenever you print. This means that if multiple users share the computer all their print jobs will authenticate with your username and password, which means all printouts will be deducted from your print quota.

If multiple users share the system, then it is best to print using one of the following methods:

Jump to ENCS servers

  1. Log into one of the ENCS servers.
  2. Print your Postscript (.ps) file using the command below. For the purpose of this example the Postscript file to print is foo.ps and the printer name is ballpoint.

    qpr -P ballpoint foo.ps

  3. Check the print queue with:

lpq -P ballpoint

One shot printing from User-Managed Linux machine

  1. Plaintext password is required for samba server. To do so, add or edit the following lines to the file /etc/samba/smb.conf in the [global] section:

    client plaintext auth = yes
    client lanman auth = yes
    workgroup = csglobal

  2. From a linux terminal use the command below. For the purpose of this example the file to print is foo.ps, the printer name is ballpoint and the ENCS username is jsmith:
    cat foo.ps | smbclient '\\printhost-um.encs.concordia.ca\ballpoint' -U jsmith -c "print -"

    It is important that the '\\printhost-um.encs.concordia.ca\ballpoint' is encased in single quotes (').

  3. When asked for your password enter your ENCS password and press ENTER.

Permanently add a printer to a User-Manged Linux machine

To add a printer in the Linux user managed system, login as root, open the terminal and type:
system-config-printer

A window pops up indicating all the existing printer configurations and status.

Select 'New' from the Printer Configuration toolbar. If you do not have any printers added yet the window will be empty, and a 'No Printers found' message box will pop up and prompt to run the printer configuration tool automatically. Click OK and select New from the Printer Configuration tool bar.

The adding printer wizard will run, follow the instructions carefully to add a new printer.

Here is a brief example

  1. To add the 'ballpoint' printer, type the queue name ballpoint and short description 'ballpoint EV8.143' and click forward.

  1. Select 'Networked Windows (SMB)' as a queue type.

    NOTE: If 'Networked Windows (SMB)' is not shown in queue type list you need to download the SMB client in order to choose this queue type. To do this, first exit the Printer Configuration by clicking on Cancel and download the SMB client using the YUM command from the command prompt:
    yum install samba-client

Then start the Printer Configuration again from step 1.

  1. Select Specify.
  2. In the Authentication dialog box, fill in the following information:

    • Workgroup: csglobal
    • Server: printhost-um.encs.concordia.ca
    • Share: ballpoint (depends on what printer you are adding), here is the list of ENCS printers.
    • User name: your ENCS user name
    • Password: your ENCS password

Remember that because you're using your ENCS username and password, printouts from all users in Linux will be deducted from your print quota. If you don't want this type of setup then read the section labeled IMPORTANT at the top of this screen.

  1. Click OK and then select Forward.
  2. The wizard will prompt for a suitable printer manufacturer and model. Depends on the printer that you are adding, choose the appropriate model. For the printer manufacturer and model, please visit this link

  1. The wizard now displays your final configuration. Click "finish" to exit the wizard.

  1. To test the new printer, a selection box will automatically pop up prompting to print a test page.

  1. Choose Yes. Check if the test page is printed correctly.
