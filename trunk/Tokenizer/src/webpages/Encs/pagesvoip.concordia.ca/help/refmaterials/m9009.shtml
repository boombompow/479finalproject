M9009 and Vista Telephone User Guide

Adjust volume of receiver

To adjust the volume of the receiver while on the phone, press the volume bar right to increase the volume of the caller’s voice or left to decrease the volume of the caller’s voice.

HOLD
If you want to place your call on hold (or continue your call at another extension of the phone), press HOLD and hang up the receiver. The hold feature keeps the caller on the line until you pick up the receiver or an extension telephone.
Note: If you receive a Call Waiting notification while a call is on hold, Call Waiting visual information will not be displayed.

USING THE CALLERS LIST (VISTA MODELS ONLY)
The CALLERS key lets you enter the Callers List. Vista 9009S model remembers up to 25 incoming calls, which are stored in the Callers List. This valuable feature helps you organize your calls by monitoring their status. Easy to understand messages and symbols keep you informed about who called, when, how often they tried to reach you, and whether you returned the call.
• Press Callers to enter the Callers List
• Press * to move through the Callers List
• Edit numbers from the dial pad by pressing _ or _
• Press _ or _ to view names and numbers
• Return a call automatically by lifting the receiver while the caller’s number is on the display or by pressing Dial
• Erase individual callers from the list by pressing # twice when the name/number is on the display. You may also erase all callers by pressing # twice when Callers List =XX is displayed
• Press Callers at any time to exit the list

To Save a caller from the Callers List:
While in the Callers List, you can save the number on the display directly into a memory key. When a name/number is displayed while in the Callers List, press Save. Press Shift and the memory key where you want to save the number in an upper memory slot. To save in a lower memory slot, press the memory key where you want to save the number. Write the name in the space provided on the memory card or receiver card.

Display Messages
• Feature is in use: Call forwarding is on
• Private number: Caller has requested number be blocked
• Unknown caller: Information on call is not available
• Long distance: Incoming call is long distance
• Message waiting Light flashes slowly: voice message waiting
• Check phone cord: Check to see if phone plugged into wall jack securely
• Ringer is OFF: Press volume bar right to turn ringer on

Display not working
Check to see if power supply is plugged in properly.

Save number of a memory key
• Press the SAVE key
• Press the memory key where you want to save the number
• Use the dial pad to enter the number. If you are entering an internal number, use the four-digit number. For external numbers, dial 9 followed by the seven-digit number.
• Press SAVE key

Erase a number on a memory key
• Press the memory key you want to erase
• Press # twice to erase the contents

Personalize your phone
There are five options available to personalize your phone.
1. Change the display language
2. Set the time and date
3. Set the ring tone and ringer volume
4. Turn the light off
Use this option to program a memory key to turn the light on and active a feature.
5. Enter your area code
Use the option when you don’t want the area code to show on local calls.

To access these options
• Without picking up the receiver, move the sliding panel below the display and press the set options [maybe you should put "set options" in italics or caps like you usually do when referring to a key]key to see the choices.
• Press * to move through the list of the five options and press the selection number for the option you want to change. Follow the prompts on the display.

Note: During power outages the telephone set will retain the time/date in memory for approximately 15 minutes. After this time you will have to reset time/date. Please note that any features or numbers programmed into the memory keys will be retained beyond 15 minutes [can we say "any features or numbers…will not be lost during a power outage" or "any features or numbers… will be retained up to 60 minutes" (or whatever it is)].

Program Call Forwarding on memory key
Memory keys can be used to activate and deactivate call forward. When call forward is ON, the feature light will be turned on.

To program Call Forwarding ON on a key
• Press SAVE
• Press 1 to program key
• Press the memory key you want to use for call forward ON
• Enter the activate call forwarding access code then # key. Enter the four-digit extension or 9 followed by the seven-digit number you forward to regularly.
• Press SAVE again

To program Call Forwarding OFF on a key
• Press SAVE
• Press 1 to program key
• Press the memory key you want to use for call forward OFF
• Enter the deactivate call forwarding access code
• Press SAVE again

Recall number of last incoming call
With the handset down, press *. The number will appear on the display.

Turn ringer OFF
With the handset down, press volume bar left until Ringer Off shows on display.

Turn ringer volume up or down
While the phone is ringing, press volume bar left for softer or right for louder.

SOME HELPFUL HINTS
NO DIAL TONE (CHECK PHONE CORD may appear on the display) Read the display. If the line cord is not attached properly, the display will advise you. Also, make sure the curly handset cord is attached securely. Telephone does not ring. (Ringer is OFF may appear on the display) The ringer may be turned OFF. Without lifting the receiver, press the right side of the volume bar to turn the ringer ON. You can adjust ringer volume and tone in the OPTIONS list.

No Information on the display and ringer volume is low.
Make sure the receiver cord is inserted securely into the jack [I think you called it a wall jack before].

When you have a power failure
The phone will operate as a regular telephone during a power failure. However, the display, the advanced features, as well as the shifted memory keys, will not operate until the power is restored.
