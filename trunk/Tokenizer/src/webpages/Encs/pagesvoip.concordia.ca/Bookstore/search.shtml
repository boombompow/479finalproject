Course Booklist

Enter a course code below to see what textbooks have been requested for it.

Enter a Department, Course Number, and Section
(ie. SPAN-200 / 51 or DESC-381 / A) to view course materials:

- /  
DEPT   -   COURSE #   /   SECTION                 

Check to show book covers (if available).

  • Leaving the Section blank will return all sections for the course (ie. all sections for ECON-201)
  • Using a partial Course Code will return all partial matches (ie. MATH-2 will return all 200-level Math courses)
  • For Continuing Education, use "CONT" as the department
  • For Marianopolis College, you may use the 3-letter department codes, and the 3-character Course/Prof Code as the Course #
  • For Queen of Angels Academy, use "QAA" as the department, and "SEC ONE" / "SEC TWO" / etc for the Course #, and leave the Section blank
  • For North Star Academy, use "NSA" as the department, and "SEC ONE" / "SEC TWO" / etc for the Course #, and leave the Section blank

Note that courses for which we have no recommended text on file will not be listed.
