Directories

The on-line and downloadable directories are provided by Instructional and Information Technology Services. The directories include faculty, staff, department and fax listings. New information is uploaded to the directory every Friday.

While the fastest method to retrieve information is with the Online Directory , for those on campus who prefer the directory in printed form, the directory may be download in PDF format.  Note: The downloadable directory is only available from within the .concordia.ca domain.

Updates to the directory may be submitted using the Directory Update form.

For Directory listings for Canada or the U.S.A., please refer to the following web sites:


Canada  Canada411.ca

             Yellowpages.ca
U.S.A.    Switchboard.com
